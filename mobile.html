spark

Spark is a music discovery app that makes finding new music fun and easy. My group and I developed it for our Object Oriented Design class at LSU. Not only did we have to create the app, but we also had to create design documentation for every feature using Enterprise Architect. My main contribution was linking the app with the Spotify API, so that a user could login automatically and cycle through songs by swiping left and right on the screen. I also implemented a feature that allowed the user to send music to a friend using SMS, and helped design the UI. We developed the app using Android Studio (for the first time) and Java.

corporation clicker

I developed Corporation Clicker solo over the summer using Unity and C#. It's an idle-clicker game for android about becoming outrageously wealthy. Developing this game was definetly one of the hardest things I've done, since I had to make almost everything myself. However, the experience showed me how chaotic development can be, and taught me how to plan around delays and bugs.
